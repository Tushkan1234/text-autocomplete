# Text Autocomplete 🤖

Учебный проект — создание модели автодополнения текста. 

## Что делает проект
Модель получает начало фразы и предсказывает её продолжение.
Например: "I'm going to" → "buy some groceries"

## Как это работает
Я обучил два варианта модели и сравнили их:
1. **LSTM** — лёгкая модель, которую можно запустить даже на телефоне
2. **distilgpt2** — мощная предобученная модель на основе трансформера

## Данные
Модель обучена на датасете **sentiment140** — это 1.6 миллиона 
реальных постов из Twitter. Тексты очищены от ссылок, упоминаний 
и спецсимволов.

## Структура проекта
- `data/` — исходный и обработанный датасет
- `src/` — весь код: обработка данных, модели, обучение, оценка
- `models/` — сохранённые веса обученной модели
- `solution.ipynb` — ноутбук с результатами и выводами

## Результаты

| Модель | ROUGE-1 | ROUGE-2 |
|--------|---------|---------|
| LSTM | 0.0000 | 0.0000 |
| distilgpt2 | 0.0377 | 0.0002 |

distilgpt2 показывает лучшее качество, однако LSTM значительно 
легче и быстрее — что важно для мобильных устройств.
